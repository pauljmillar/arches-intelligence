# Arches Intelligence

AI-Powered Conversational Analytics for McDonald's Market GMs — prototype demo for the EDAA transformation role.

Ask plain-English questions about market performance (comp sales, loyalty, drive-thru, weather impact, order accuracy) and get synthesized answers with confidence indicators, data source attribution, mini-charts, and follow-up question chips.

## Stack

React (Vite) · Recharts · vanilla CSS with brand tokens · scripted responses (no live LLM).

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Auto-deploys to Vercel from `main`.
